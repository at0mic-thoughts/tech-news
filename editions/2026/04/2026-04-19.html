🧠 Anthropic Ships Claude Opus 4.7

Anthropic released Claude Opus 4.7 on April 16, bringing a 13% improvement on coding benchmarks over Opus 4.6, better high-resolution image understanding, and improved instruction following on long-running tasks. 🤖 OpenAI Codex Goes Full Superapp

OpenAI dropped a massive Codex update on April 16, transforming it from a coding assistant into something closer to a general-purpose agentic desktop app. Key additions: background computer use (see your screen, click, type with its own cursor, multiple agents in parallel on macOS), an in-app browser, image generation via gpt-image-1.5, 90+ new plugins (JIRA, CircleCI, GitLab, Neon), SSH to remote devboxes, and a preview of persistent memory. The wildest feature: Codex can now schedule future work for itself and wake up automatically to continue tasks across days. 3 million weekly developers and growing.

🔧 OpenAI Agents SDK Gets Native Sandbox Support

On April 15, OpenAI shipped a major update to its Agents SDK, adding native sandbox execution as a first-class primitive. Developers can now isolate agent workloads through Cloudflare, Vercel, and E2B. The update also includes new tools for multi-agent orchestration, handoffs, and safety guardrails. "Can this agent run safely and recoverably over many steps?" is now answered at the SDK level rather than left to individual developers. Infrastructure-level plumbing that matters more than any individual model release for production agent builders.
